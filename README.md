Cara menjalankan menggunakan [xampp](https://www.apachefriends.org/):
 -Letakan konten dalam Pertemuan-12-Program-CRUD-Nizar pada folder xampp/htdocs lalu jalankan xampp dengan Apache dan MySQL aktif
 -Buka halaman [loclhost/phpMyAdmin](http://localhost/phpmyadmin)
 -Upload database.sql 
 -Buka localhost/Pertemuan12Nomor1dan2/index.php untuk mengakses tugas nomor 1 dan 2
 -Buka localhost/Pertemuan12Nomor3/konversi_nilai.php untuk mengakses tugas nomor 3.1 form konversi nilai
 -Buka localhost/Pertemuan12Nomor3/pendataan_mahasiswa.php untuk mengakses tugas nomor 3.2 form pendataan mahasiswa
 -Atau buka localhost/navigator.php untuk mengakses tugas menggunakan folder navigator
